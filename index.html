BoxedWine is an emulator that runs Windows applications. It achieves this by running an unmodified 32-bit version of Wine, and emulating the Linux kernel and CPU.

Click on "Start" and Import your own applications into BoxedWine and explore it through Wine's own Explorer application!

Start

Currently BoxedWine only supports up to Wine 5.0 so that's what we went with and also this is for people that want to run their own apps as the official website doesn't allow running your own apps easily.
